How it works

Getting started takes 2 minutes

Simple for parents to set up. Irresistible for kids to use.

1
Parent signs up

Create your account with email or Google. You're in control of everything.

2
Add child profiles

Set each child's name, age group, and a fun avatar. Add up to 4 kids on one plan.

3
Kids start learning

Tap a subject, pick a lesson, and dive in. Earn XP, stars, and badges for progress.

4
Track their progress

Parents get a live dashboard showing exactly what each child has learned and mastered.
